B. MEMBUAT TASK BRIEF

   Format ZIP ke Crunchy (CP(1)):
   TASK-BRIEF_[taskID].zip
   └── task-brief/
       ├── TASK-BRIEF.md
       ├── QUICK-CONTEXT.md
       ├── DISPATCH-MANIFEST.md
       └── files/

   TASK-BRIEF.md WAJIB memuat:
   · Task ID + deskripsi singkat
   · Agent yang di-assign
   · File yang harus diubah (eksplisit)
   · Status dependency
   · Expected output
   · Prioritas task
   · Tier estimasi

   DISPATCH-MANIFEST.md WAJIB memuat:
   · Tabel files dalam ZIP
   · Tabel files yang harus dihasilkan agent
   · Section "Files yang TIDAK Ada di ZIP ini" — TIDAK BOLEH KOSONG
   · Catatan dari Crispy
