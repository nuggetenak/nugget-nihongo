# Nugget Nihongo — Scripts

## Architecture

Data lives in **category files** (small, editable, auditable).  
Monolithic files are **auto-generated** — never edit them directly.

```
public/data/vocab/
  n5/n5-verbs.js          ← EDIT THESE
  n5/n5-nouns.js
  ...
  vocab-n5.js             ← AUTO-GENERATED by merge-vocab.js

public/data/grammar/
  n3/n3-modality.js       ← EDIT THESE
  n3/n3-connectives.js
  ...
  grammar-n3.js           ← AUTO-GENERATED by merge-grammar.js
```

## Scripts

| Script | What it does |
|--------|-------------|
| `node scripts/merge-vocab.js` | Rebuilds all `vocab-nX.js` from category files |
| `node scripts/merge-grammar.js` | Rebuilds all `grammar-nX.js` from category files |
| `python3 scripts/split-vocab.py` | (Run once) Splits monolithic → category files |
| `python3 scripts/split-grammar.py` | (Run once) Splits monolithic → category files |
| `python3 scripts/jmdict-enrich.py` | Enrich entries with JMDict data |

## Workflow for adding new entries

1. Edit the right category file, e.g. `public/data/vocab/n3/n3-verbs.js`
2. Run `node scripts/merge-vocab.js`
3. Run `node tests/run.js` — must be 0 FAIL
4. Run `node tests/quality.js --new-only --errors-only` — must be 0 errors
5. Bump version in `version.js` and `sw.js`
6. `git add -A && git commit && git push origin develop`

## Category structure

### Vocab categories (per level)
- `verbs` — verb-u, verb-ru, verb-suru, verb-kuru
- `adjectives` — i-adj, na-adj, no-adj
- `adverbs` — adverbs, conjunctions
- `nouns` — noun, noun-temporal, proper-noun, counter, numeral
- `expressions` — expressions, particles, interjections

### Grammar categories (per level)
- `tense-aspect` — past, progressive, sequential, completion
- `verb-forms` — potential, passive, causative, volitional
- `conditionals` — tara, ba, nara
- `modality` — conjecture, hearsay, obligation, permission
- `negation-extent` — negative, comparison, extent
- `connectives` — reason, contrast, listing, purpose, quotation
- `nouns-predicates` — nominalization, copula, existence
- `desire-social` — want, request, keigo patterns
- `particles` — particle usage patterns
- `adverbs` — adverbial patterns
- `expressions` — set expressions, interrogative

## JMDict enrichment

JMDict (216,568 entries) is available at `/tmp/jmdict_lookup.pkl`.

```bash
# Dry run (shows what would change without writing)
python3 scripts/jmdict-enrich.py --dry-run --verbose --level n5

# Run for a specific level
python3 scripts/jmdict-enrich.py --level n4

# Run for all levels, then rebuild
python3 scripts/jmdict-enrich.py
node scripts/merge-vocab.js
```
